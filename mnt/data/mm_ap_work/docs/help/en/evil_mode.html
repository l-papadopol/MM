MM Flow Studio / AutoQSO

MM Flow Studio is the visual automation environment. The editor supports Event, Decision, Action, Timer, Variable, Compare, Loop, I/O, Math and End blocks, with a vertical toolbar on the left.

The graph is executed by a runtime connected to MadModem’s event bus. Actions pass through the Capability Manager: no direct PTT, TX audio, CAT or rotator control from the graph.
