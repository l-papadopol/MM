MM Flow Studio / AutoQSO

MM Flow Studio je vizuální automatizační prostředí s bloky Event, Decision, Action, Timer, Variable, Compare, Loop, I/O, Math a End, svislá lišta je vlevo.

Graf provádí runtime napojený na event bus MadModem. Akce jdou přes Capability Manager: žádné přímé PTT, TX audio, CAT ani rotátor z grafu.
