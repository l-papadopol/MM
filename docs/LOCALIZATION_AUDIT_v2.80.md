# Localization audit — v2.80

A full translation dictionary harvest was run over the source tree and Qt Designer forms. The runtime dictionaries now contain 1162 keys per language:

- English
- Italian
- French
- German
- Norwegian
- Czech

The v2.80 pass focused on configuration/menu text that was still visible in English after selecting Italian, especially mode settings pages and the new scheduler dialog.

Important limitation: this is a source/dictionary audit, not a native-speaker proofread of every generated fallback string. Technical protocol names and standard radio abbreviations intentionally remain unchanged.
