CW / Morse

CW verwendet die integrierte Skimmer-Engine, behält aber die manuelle A/B-Abstimmung im Wasserfall. Linksklick wählt RX A, Rechtsklick den optionalen RX B. CW-TX erzeugt einen sauberen Sinuston mit Raised-Cosine-Tastung und schmalem Sendebandpass.

CW-Empfang: CW RX verwendet die integrierte Skimmer-Engine. Linksklick setzt den grünen RX-A-Marker; Rechtsklick setzt den optionalen blauen RX-B-Marker. Nur A/B werden in das RX-Terminal übernommen; andere Dekodierungen können als temporäre OSD-Labels im Wasserfall erscheinen. MIND wird nicht für CW-Text verwendet.
