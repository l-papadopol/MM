Logbuch und Karte

QSO Map nutzt einen einzigen Layers...-Knopf für OSM/Fallback, Maidenhead-Gitter, Home→QSO-Linien, Marker und Filter.

Im Reiter Rotor kann ein Maidenhead-Locator, Land/DXCC oder Präfix aus cty.csv eingegeben werden. MM berechnet Peilung und Entfernung vom QTH und richtet den aktiven Rotor aus.

Der ADIF-Export enthält Optionen für ein genaues UTC-Zeitintervall. Aktivieren Sie den TIME_ON-Filter, setzen Sie Start und Ende als UTC-Datum/Zeit, und MM exportiert nur QSOs mit Start <= TIME_ON < Ende. Das ist nützlich für kurze Contests und Activity Days, zum Beispiel einen vierstündigen FT8 Activity Day.
