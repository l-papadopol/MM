Loggbok og kart

QSO Map bruker én Layers...-knapp for OSM/fallback-bakgrunn, Maidenhead-rutenett, Home→QSO-linjer, markører og filtre.

I Rotor kan du skrive Maidenhead-locator, land/DXCC eller prefiks fra cty.csv. MM beregner bearing og avstand fra QTH og peker aktiv rotor.

ADIF-eksporten har nå alternativer for et presist UTC-tidsintervall. Aktiver TIME_ON-filteret, sett start og slutt som UTC dato/tid, og MM eksporterer bare QSOer med start <= TIME_ON < slutt. Dette er nyttig for korte konkurranser og activity days, for eksempel en fire timers FT8 Activity Day.
