MM Flow Studio

Flow Studio bruker Event Bus og Capability Manager. Flow-blokker styrer ikke PTT, audio, CAT eller rotor direkte; de ber om sikre handlinger.

Mottakstabellen viser Bearing. FT TX styres av UTC-scheduler og sperres hvis stasjonsidentitet mangler eller rotoren ikke er klar.
