MM Flow Studio / AutoQSO

MM Flow Studio er det visuelle automasjonsmiljøet med blokker for Event, Decision, Action, Timer, Variable, Compare, Loop, I/O, Math og End, med vertikal verktøylinje til venstre.

Grafen kjøres av en runtime koblet til MadModems eventbus. Handlinger går via Capability Manager: ingen direkte PTT, TX-audio, CAT eller rotor fra grafen.
