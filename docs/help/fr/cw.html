CW / Morse

Le CW utilise le moteur skimmer intégré tout en gardant le réglage manuel A/B sur la waterfall. Clic gauche pour RX A, clic droit pour RX B optionnel. Le TX CW génère une sinusoïde propre avec mise en forme raised-cosine et filtrage passe-bande étroit.

Réception CW : CW RX utilise le nouveau moteur skimmer intégré. Clic gauche pour le marqueur vert RX A ; clic droit pour le marqueur bleu optionnel RX B.
