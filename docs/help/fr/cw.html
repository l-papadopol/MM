CW / Morse

La CW utilise un waterfall compact car le skimmer a été retiré. L’accord RX CW se fait avec le marqueur de tonalité sur le waterfall. AFC peut suivre de petites dérives, et Software AGC se trouve maintenant dans les réglages du mode CW sous AFC pour le QSB ou les niveaux d’entrée instables. Le TX CW utilise un sinus propre, une mise en forme raised-cosine et un passe-bande étroit pour éviter le splatter.

Récupération CW manuel : la réception CW utilise maintenant un suivi WPM robuste à fenêtre mobile, une détection adaptative des espacements lettre/mot, le filtrage ggmorse de la note sélectionnée et une normalisation légère des tokens radio.
