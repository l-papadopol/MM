CW / Morse

La CW utilise un waterfall compact car le skimmer a été retiré. L’accord RX CW se fait avec le marqueur de tonalité sur le waterfall. AFC peut suivre de petites dérives. Le TX CW utilise un sinus propre, une mise en forme raised-cosine et un passe-bande étroit pour éviter le splatter.

Réception CW : CW RX utilise le décodeur ggmorse intégré comme unique source de texte. La note choisie, la largeur de bande, l’AFC et les marqueurs RX A/B préparent le signal ; MIND ne produit pas de texte CW.
