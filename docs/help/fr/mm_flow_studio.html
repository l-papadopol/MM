MM Flow Studio

Flow Studio utilise Event Bus et Capability Manager. Les blocs ne pilotent pas directement PTT, audio, CAT ou rotor: ils demandent des actions sûres aux modules réels.

La table des reçus inclut le Bearing. Le TX FT reste sous le scheduler UTC et est inhibé si l’identité station manque ou si le rotor demandé n’est pas prêt.
