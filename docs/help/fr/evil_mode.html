MM Flow Studio / AutoQSO

MM Flow Studio est l’environnement visuel d’automatisation. Il supporte blocs Événement, Décision, Action, Timer, Variable, Comparaison, Boucle, I/O, Math et Fin, avec barre verticale à gauche.

Le graphe est exécuté par un runtime connecté au bus d’événements MadModem. Les actions passent par le Capability Manager : pas de PTT, audio TX, CAT ou rotateur direct depuis le graphe.
