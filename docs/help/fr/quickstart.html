Démarrage rapide

  1. Utilisez Paramètres → Paramètres... pour Audio/PTT/CAT, Utilisateur/QTH/Macros, Journal/couleurs et filtres FT, Planificateur et calibration carte son. Les anciennes fenêtres séparées sont regroupées en onglets.
  2. Sélectionnez le mode, puis utilisez le petit bouton ▶ RX en haut de l’onglet Statut.
  3. Utilisez le bouton ● TX voisin seulement quand le mode courant dispose d’un contenu TX valide; les modes FT gardent leurs contrôles TX sûrs par créneau dans l’onglet Mode.
  4. Gardez un niveau audio propre, sans écrêtage.
  5. Cliquez le marqueur du waterfall et testez TX seulement après vérification PTT/ALC.
  6. Pour les diagnostics FT4/FT8, ouvrez le journal d’exécution depuis l’onglet Mode.
