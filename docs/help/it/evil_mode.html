MM Flow Studio / AutoQSO

MM Flow Studio è l’ambiente visuale di automazione. L’editor permette blocchi Evento, Decisione, Azione, Timer, Variabile, Confronto, Loop, I/O, Matematica e Fine, con toolbar verticale a sinistra.

Il grafo viene eseguito da un runtime collegato all’event bus di MadModem. Le azioni passano attraverso il Capability Manager: niente PTT, audio TX, CAT o rotore comandati direttamente dal grafo.
