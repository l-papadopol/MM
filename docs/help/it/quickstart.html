Primo avvio

  1. Configura Audio / PTT / CAT nella finestra Impostazioni.
  2. Seleziona il modo operativo, poi usa il piccolo pulsante ▶ RX in alto nella tab Stato.
  3. Usa il pulsante ● TX affiancato solo quando il modo corrente ha contenuto TX valido; nei modi FT i controlli TX sicuri a slot restano nella tab Modo.
  4. Regola il livello audio senza clipping.
  5. Clicca il waterfall per posizionare il marker del modo.
  6. Prova TX solo dopo aver verificato PTT e livello ALC.
  7. Per la diagnostica FT4/FT8, apri Log runtime dalla tab Modo.
