CW / Morse

CW používá integrovaný skimmer, ale ruční ladění A/B ve waterfallu zůstává. Levé kliknutí vybere RX A, pravé kliknutí volitelný RX B. CW TX generuje čistý sinus s raised-cosine klíčováním a úzkým vysílacím pásmem.

Příjem CW: CW RX používá integrovaný skimmer. Levé kliknutí nastaví zelený marker RX A; pravé kliknutí volitelný modrý marker RX B. Do RX terminálu se posílají pouze A/B; ostatní dekódování se mohou zobrazit jako dočasné OSD popisky ve waterfallu. MIND se pro CW text nepoužívá.
