Deník a mapa

QSO Map používá jediné tlačítko Layers... pro OSM/fallback, Maidenhead síť, cesty Home→QSO, značky a filtry.

V Rotátor lze zadat Maidenhead lokátor, zemi/DXCC nebo prefix z cty.csv. MM spočítá azimut a vzdálenost z QTH a natočí aktivní rotátor.

Export ADIF nyní obsahuje volby pro přesný časový interval UTC. Zapněte filtr TIME_ON, nastavte začátek a konec jako UTC datum/čas a MM exportuje pouze QSO s začátek <= TIME_ON < konec. Hodí se to pro krátké závody a activity days, například čtyřhodinový FT8 Activity Day.
