First run

  1. Open Settings and configure Audio / PTT / CAT.
  2. Select a mode, then use the small ▶ RX button at the top of the Status tab.
  3. Use the adjacent ● TX button only when the current mode has valid TX content ready; FT modes keep their slot-safe TX controls in Mode.
  4. Keep audio level clean and below clipping.
  5. Click the waterfall to place the active marker.
  6. Test TX only after PTT and ALC are safe.
  7. For FT4/FT8 diagnostics, open Runtime log from the Mode tab.
