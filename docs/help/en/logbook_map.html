Logbook and map

QSO Map uses one Layers... button for OSM/fallback background, Maidenhead grid, Home→QSO paths, markers and filters. Old duplicate controls were removed.

In the Rotator tab you can type a Maidenhead locator, country/DXCC or prefix from cty.csv. MM computes bearing and distance from your QTH and points the active rotator.

ADIF export now includes export options for a precise UTC time interval. Enable the TIME_ON filter, set start and end as UTC date/time, and MM exports only QSOs with start <= TIME_ON < end. This is useful for short contests and activity days, for example a four-hour FT8 Activity Day.
