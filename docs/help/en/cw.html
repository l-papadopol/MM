CW / Morse

CW uses a compact waterfall because the skimmer was removed. CW RX tuning is done from the waterfall tone marker. AFC can follow small drift, and Software AGC is now in the CW Mode settings under AFC for QSB or unstable input levels. CW TX generates a clean sine, raised-cosine key shaping and a narrow band-pass around the tone to avoid harmonics and splatter.

Human fist recovery: CW RX now uses robust moving-window WPM tracking, adaptive letter/word gap detection, selected-note ggmorse filtering and light radio-token spacing normalization for hand-sent Morse.
