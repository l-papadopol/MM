Radio Telescope

Radio Telescope is a receive-only mode that maps relative radio-noise power across the visible Alt-Az sky. It commands a configured rotator and stores one integrated measurement for each beam-sized hexagonal cell.

Before scanning

  1. Configure and test the Alt-Az rotator profile, limits, rest position and emergency stop.
  2. Enter the antenna beam width in the rotator profile; it determines the hexagonal sampling size.
  3. Choose lower and upper audio frequencies, settling time and integration time.
  4. Confirm that the receiver, audio input and selected band are correct.

Running a scan

Start Scan first returns the antenna to the required initial position when necessary. The controller points to the centre of each cell, waits for mechanical settling, integrates the selected audio band and records the result. Endpoint handling, direction reversal and return to the pre-scan position depend on the active rotator profile and scan settings.

Map and data

The live map uses semi-transparent adjacent hexagons over the degree grid. Measurements are normalized on a logarithmic scale for the colour map. Use the sample list and CSV export for numerical analysis; Reset Map clears the current scan data.

Safety

The mode does not transmit, but it can move a large antenna system automatically. Never start a scan until azimuth/elevation limits, cable wrap, obstructions and an accessible stop method have been verified. When no rotator is connected, MadModem must report the condition instead of pretending that a physical scan occurred.
